Detailed Description

The ClimateIoT class is a class for connecting the Climate Card to the IoT module.

This function allows you to control the Climate Card using MQTT.

Warning
You should not instantiate this class directly, instead use ESPMegaIoT's registerCard function.

Initializes the ClimateIoT component.

This function sets the MQTT client, base topic, card ID, and card pointer. It also registers the sensor and air conditioner update callbacks.

Parameters

cardA pointer to the ExpansionCard object.
mqttA pointer to the PubSubClient object.
base_topicThe base topic for MQTT communication.
Returns
True if the initialization is successful, false otherwise.
