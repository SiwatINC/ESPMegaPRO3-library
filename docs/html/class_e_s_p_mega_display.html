Detailed Description

The ESPMegaDisplay class is a class for controlling the ESPMegaDisplay.

Note
The ESPMegaDisplay is a UART controlled display.
Connect the Display's TX pin to the ESPMega's RX pin and the Display's RX pin to the ESPMega's TX pin.
