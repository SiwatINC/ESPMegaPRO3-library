Detailed Description

The DigitalInputIoT class is a class for connecting the Digital Input Card to the IoT module.

This function allows you to control the Digital Input Card using MQTT.

Warning
You should not instantiate this class directly, instead use ESPMegaIoT's registerCard function.

Initializes the DigitalInputIoT object.

This function sets the necessary parameters for the DigitalInputIoT object, such as the card ID, expansion card, MQTT client, and base topic. It also enables the publishing of digital input values and registers a callback function for handling value changes.

Note
Although this function can be called in the main program, it is recommended to use ESPMegaIoT::registerCard() to automatically manage the instantiation and initialization of this component.
Parameters
card_idThe ID of the card.
cardPointer to the DigitalInputCard object.
mqttPointer to the PubSubClient object.
base_topicThe base topic for MQTT communication.
Returns
True if the initialization is successful, false otherwise.
