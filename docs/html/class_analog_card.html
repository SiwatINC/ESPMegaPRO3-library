Detailed Description

This class represents the Analog Card.

The analog card has 8 analog inputs accross two banks, and 4 DAC outputs.

Note
You do not need to specify the ESPMega I/O Address when creating an instance of this class as there can only be one Analog Card installed in the ESPMegaPRO board.
Warning
There can only be one Analog Card installed in the ESPMegaPRO board.
