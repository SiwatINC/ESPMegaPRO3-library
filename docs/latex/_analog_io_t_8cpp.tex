\doxysubsection{Detailed Description}
Implementation of the \mbox{\hyperlink{class_analog_io_t}{Analog\+IoT}} class. 

This file contains the implementation of the \mbox{\hyperlink{class_analog_io_t}{Analog\+IoT}} class, which provides functionality for handling analog input and output operations in an IoT system. The class allows for setting the state and value of digital-\/to-\/analog converters (DACs), as well as reading the value of analog-\/to-\/digital converters (ADCs). It also supports publishing the state and value of DACs and ADCs over MQTT.
