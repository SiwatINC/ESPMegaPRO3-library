\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_e_s_p_mega_display}{ESPMega\+Display}} class is a class for controlling the \mbox{\hyperlink{class_e_s_p_mega_display}{ESPMega\+Display}}. 

\begin{DoxyNote}{Note}
The \mbox{\hyperlink{class_e_s_p_mega_display}{ESPMega\+Display}} is a UART controlled display. 

Connect the Display\textquotesingle{}s TX pin to the ESPMega\textquotesingle{}s RX pin and the Display\textquotesingle{}s RX pin to the ESPMega\textquotesingle{}s TX pin. 
\end{DoxyNote}
