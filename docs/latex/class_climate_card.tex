\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_climate_card}{Climate\+Card}} class is a class for controlling an air conditioner. 

This class allows you to control an air conditioner using an IR LED. It is meant to be used with the ESPMega Climate Card.

\begin{DoxyNote}{Note}
You can also use a DHT22 or DS18\+B20 temperature sensor to get the room temperature (and humidity if using a DHT22). Although, this is optional. 
\end{DoxyNote}
