\doxysubsection{Detailed Description}
The internal display of the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}}. 

This is the display that is installed on some \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} Chassis. It is a 3.\+5"{} TFT LCD with a resistive touch screen.

You can use this display to monitor the status of the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} and also to control the various components of the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}}.

If you are using a custom display, you need to create a class that inherits from \mbox{\hyperlink{class_e_s_p_mega_display}{ESPMega\+Display}} and implement the methods in that class, you may refer to this class for reference.

\begin{DoxyNote}{Note}
This class is automatically instantiated by the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} and can be accessed via the {\ttfamily display} variable. 
\end{DoxyNote}
