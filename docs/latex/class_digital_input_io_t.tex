\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_digital_input_io_t}{Digital\+Input\+IoT}} class is a class for connecting the Digital Input Card to the IoT module. 

This function allows you to control the Digital Input Card using MQTT.

\begin{DoxyWarning}{Warning}
You should not instantiate this class directly, instead use \mbox{\hyperlink{class_e_s_p_mega_io_t}{ESPMega\+IoT}}\textquotesingle{}s register\+Card function. 
\end{DoxyWarning}

It also enables the publishing of digital input values and registers a callback function for handling value changes.
