\doxysubsection{Detailed Description}
The base class for all expansion cards. 

In order to create a new expansion card, you should create a new class that inherits from this class. Your class should implement the following functions\+:
\begin{DoxyItemize}
\item begin() \+: Initialize the card
\item loop() \+: A function that is called in the main loop
\item get\+Type() \+: Get the type of the card, The type should be a unique number between 0 and 255
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
This class is abstract and should not be instantiated directly.
