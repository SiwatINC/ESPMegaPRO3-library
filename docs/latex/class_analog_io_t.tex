\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_analog_io_t}{Analog\+IoT}} class is a class for connecting the Analog Card to the IoT module. 

This function allows you to control the Analog Card using MQTT.

\begin{DoxyWarning}{Warning}
You should not instantiate this class directly, instead use \mbox{\hyperlink{class_e_s_p_mega_io_t}{ESPMega\+IoT}}\textquotesingle{}s register\+Card function. 
\end{DoxyWarning}
