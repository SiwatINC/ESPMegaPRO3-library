\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_digital_output_card}{Digital\+Output\+Card}} class is a class for controlling the Digital Output Card. 

This Digital Output Card has 16 digital outputs. All outputs are PWM capable. ALl outputs are 12V Push-\/\+Pull outputs. Outputs is grouped in 4 groups of 4 outputs.(0-\/3, 4-\/7, 8-\/11, 12-\/15) Each pin is capable of 0.\+6A, however each group\textquotesingle{}s total current is limited to 1.\+2A.

\begin{DoxyWarning}{Warning}
You should not instantiate this class directly, instead use ESPMega\+IO\textquotesingle{}s register\+Card function. 
\end{DoxyWarning}
