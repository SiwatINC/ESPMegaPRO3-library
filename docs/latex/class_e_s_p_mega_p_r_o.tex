\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} class is the main class for the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} library. 

This class provides functions for managing the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} board, such as installing expansion cards, managing the internal RTC, and managing the internal FRAM. This class also provides functions for managing the \mbox{\hyperlink{class_e_s_p_mega_io_t}{ESPMega\+IoT}} module and the internal display.

This class provide a Object Oriented Programming (OOP) interface for the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} board. If you are looking for a more simple and a more procedural interface, please use the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} class in ESPMega\+PRO.\+hpp. But note that the \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} class only interfaces with the built-\/in Digital Input and Digital Output cards and other onboard components. It does not provide an interface for expansion cards, the \mbox{\hyperlink{class_e_s_p_mega_io_t}{ESPMega\+IoT}} module, and the internal display.

\begin{DoxyWarning}{Warning}
Only one \mbox{\hyperlink{class_e_s_p_mega_p_r_o}{ESPMega\+PRO}} object can be created, creating more than one will result in undefined behavior. 
\end{DoxyWarning}
